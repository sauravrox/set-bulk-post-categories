Allows user to set the desired categories as well as unset the categories of all the posts in a bulk without editing the posts itself.

== Description ==

You can set & unset the categories of the posts in a single screen. Allows user to set the desired categories as well as unset the categories of all the posts in a bulk without editing the posts itself. Hierarchy of the respective category is shown in the light-grey color at the left side of the category which helps to understand if it is Parent or Child category. You can also Edit/Delete the posts at the same screen.

== Installation ==

= From your WordPress dashboard =

1. Visit 'Plugins > Add New'
2. Search for 'set-bulk-post-categories'
3. Activate Set Bulk Post Categories plugin from your Plugins page. 
4. A menu will be added at the admin panel called 'Set Bulk Post Categories'. Click on it & you will see the desired plugin result.

= From WordPress.org =

1. Download set-bulk-post-categories.
2. Upload the 'set-bulk-post-categories' directory to your '/wp-content/plugins/' directory, using your favorite method (ftp, sftp, scp, etc...)
3. Activate Set Bulk Post Categories from your Plugins page. 
4. A menu will be added at the admin panel called 'Set Bulk Post Categories'. Click on it & you will see the desired plugin result.

== Frequently Asked Questions ==

= How does it work? =

After activation, a menu 'Set Bulk Post Categories' is added on the Admin panel. When clicked, shows all the posts with its title as well as its categories on the left. You can set & unset the categories of corresponding posts according to  requirement. Hit the 'Submit' button and you are done.

= What is shown in the 'Set Bulk Post Categories' screen? =

It shows post title as well as the all the available categories. Checkbox is checked if that post is currently categorized to that category. Hierarchy of the categories is shown in light-grey color which helps to understand if the category is Parent or a Child.
